# 📚 我的书柜 - 经典文学收藏

一个精美的交互式书柜展示网页，采用现代设计风格，展示经典文学作品。点击书籍可查看详情，支持收藏和分享功能。

## ✨ 新特性

### 🎨 全新视觉设计
- **现代莫兰迪色系** - 温暖优雅的配色方案
- **玻璃拟态效果** - 半透明毛玻璃质感
- **精美书籍设计** - 逼真的书脊纹理和装饰
- **柔和阴影** - 层次分明的光影效果

### 📱 响应式布局
- 完美适配桌面、平板和手机
- 触摸友好的交互设计
- 优化的移动端体验

### ❤️ 收藏功能
- 点击心形按钮收藏喜爱的书籍
- 数据自动保存到本地
- 收藏状态实时同步

### 🔄 分享功能
- 一键分享书籍到社交媒体
- 自动生成精美分享文案
- 支持 Web Share API 和剪贴板复制

### 📊 数据统计
- 实时显示书籍总数
- 展示作者数量
- 统计文学类型

### 🎭 动画效果
- 流畅的书籍入场动画
- 优雅的悬停效果
- 平滑的模态框过渡
- 植物装饰摆动动画

## 🚀 使用方法

### 本地预览

直接在浏览器中打开 `index.html` 文件即可。

### 使用本地服务器（推荐）

**Python:**
```bash
cd bookshelf
python3 -m http.server 8080
```

**Node.js:**
```bash
cd bookshelf
npx http-server -p 8080
```

然后访问 http://localhost:8080

## 🎯 交互方式

1. **浏览书架** - 鼠标悬停或触摸书籍查看效果
2. **打开书籍** - 点击书籍查看详细信息
3. **收藏书籍** - 点击详情页的「收藏」按钮
4. **分享书籍** - 点击详情页的「分享」按钮
5. **关闭详情** - 点击关闭按钮、背景或向下滑动（移动端）

## ⌨️ 快捷键

- `ESC` - 关闭书籍详情

## 🔧 扩展功能

### 添加新书籍

```javascript
BookshelfAPI.addBook({
    title: "新书名",
    author: "作者名",
    year: "2024",
    genre: "类型",
    description: "书籍描述",
    size: "medium",  // small, medium, large
    color: 1         // 1-6
});
```

### API 接口

- `BookshelfAPI.search(query)` - 搜索书籍
- `BookshelfAPI.filterByGenre(genre)` - 按类型筛选
- `BookshelfAPI.getGenres()` - 获取所有类型
- `BookshelfAPI.getFavorites()` - 获取收藏的书籍
- `BookshelfAPI.openBook(book)` - 打开指定书籍
- `BookshelfAPI.closeBook()` - 关闭详情

## 🛠 技术栈

- **HTML5** - 语义化结构

- **Vanilla JavaScript** - 原生 JavaScript，无依赖
- **LocalStorage** - 本地数据存储
- **Web Share API** - 系统分享功能
